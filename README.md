# 🌍 LearnEarth — 地球文化百科

> 深度了解地球上每一个国家的文化，拓展视野，开放意识，多元视角。

## 项目目标

这个项目不是地理教科书，也不是旅行攻略。它的目标是帮助你**真正理解**不同文化中的人如何看待世界——他们的价值观、恐惧、骄傲、禁忌、笑点和泪点。

读完一个国家的文件，你应该能够：
- 理解为什么这个国家的人会做出某些让外人困惑的行为
- 识别并挑战自己对这个国家的刻板印象
- 用这个国家人民自己的视角来看待他们的历史和现实
- 找到与这个文化深入接触的入口（书、电影、音乐）

## 每个国家文件包含

1. **基本概况** — 地理、人口、语言、宗教
2. **历史脉络** — 塑造当代文化的关键历史事件
3. **核心价值观与世界观** — 这个民族如何理解个人、集体、时间、权威
4. **社会结构** — 家庭、阶级、性别、权力关系
5. **日常生活与礼仪** — 习俗、禁忌、待客之道
6. **饮食文化** — 食物背后的文化意义
7. **艺术、音乐与文学** — 这个文化的精神表达
8. **节庆与仪式** — 重要节日与人生仪式
9. **常见误解 vs 真实面貌** — 挑战刻板印象
10. **与其他文化的交汇** — 历史交融与现代影响
11. **深入了解的资源** — 书单、片单、音乐

## 学习建议

- **不要只读熟悉的国家** — 最大的认知扩展往往来自陌生的文化
- **带着问题读** — 读之前问自己："我对这个国家有什么假设？"
- **读完后找一部该国电影看** — 文字描述 + 视觉体验效果翻倍
- **记录你的反应** — 什么让你感到惊讶？什么挑战了你的假设？
