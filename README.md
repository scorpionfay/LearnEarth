# 🌍 LearnEarth — 地球文化百科

> 深度了解地球上每一个国家的文化，拓展视野，开放意识，多元视角。

---

## 项目目标

这个项目不是地理教科书，也不是旅行攻略。它的目标是帮助你**真正理解**不同文化中的人如何看待世界——他们的价值观、恐惧、骄傲、禁忌、笑点和泪点。

读完一个国家或主题的文件，你应该能够：
- 理解为什么这个国家的人会做出某些让外人困惑的行为
- 识别并挑战自己的刻板印象，用当地人的视角重新审视历史
- 将历史模式（帝国兴衰、革命逻辑、殖民遗产）与今日世界联系起来
- 找到深入了解某个文化或议题的入口（书单、片单、延伸阅读）

## 学习建议

- **不要只读熟悉的国家** — 最大的认知扩展往往来自最陌生的文化
- **带着问题读** — 读之前问自己："我对这个国家/议题有什么假设？"
- **横向阅读** — 读完某国文件，再去 Patterns/ 看相关历史规律
- **读完后找一部该国电影看** — 文字描述 + 视觉体验效果翻倍
- **记录你的反应** — 什么让你感到惊讶？什么挑战了你的假设？

## 内容原则

每篇文章都力求：
- **多元视角**：不只讲胜利者的故事，也讲被压迫者、少数群体和历史受害者的视角
- **批判性思维**：呈现争议，而非给出唯一答案
- **历史深度**：理解当下需要理解过去
- **延伸阅读**：每篇文章末尾提供书单，供深入探索
